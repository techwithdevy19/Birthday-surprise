# 🎂 Birthday Wishes Website

## Files Included
- `index.html`    — Main HTML structure
- `style.css`     — All styles (responsive for mobile & desktop)
- `script.js`     — Core logic, personalization, sharing
- `confetti.js`   — Confetti celebration engine
- `particles.js`  — Ambient sparkle particle background

## Assets to Add (you provide these)
- `bg.mp4` or `bg.png` — Background video/image (plays blurred in loop)
- `gif1.png` — Gallery GIF 1
- `gif2.png` — Gallery GIF 2
- `gif3.png` — Gallery GIF 3
- `gif4.png` — Gallery GIF 4

## How to Use
1. Open `index.html` in any browser
2. Enter the birthday person's name and date of birth
3. Enjoy the celebration! 🎉

## Features
- Onboarding screen (name + DOB)
- Hero with animated name & age badge
- 4-image GIF gallery
- Personalized love note (editable & saved)
- 9 birthday wish cards
- WhatsApp & Telegram sharing
- Confetti burst on demand
- Ambient particle background
- Responsive for mobile & desktop
- URL param support (share pre-filled link)
